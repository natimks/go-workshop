# go-workshop
Workshop Golang

# Objetivo
Criação de um CRUD em que é possível: inserir, atualizar, deletar e consultar usuários e dívidas.
Cada usuário do sistema pode ter diversas dívidas.

# Framework & Libs

Gin-gonic, Gorm, Postgres
